**MANAGE YOUR DATA, NOT THEIR STRING REPRESENTATION!**

**react-text-renderer-components** is a zero-dependencies component library providing a set of (pure) text rendering utility components.

## Components

- [DatenRenderer](#daterenderer)
- [TimeRenderer](#timerenderer)
- [DateTimeRenderer](#datetimerenderer)
- [QuarterRenderer](#quarterrenderer)
- [WeekRenderer](#weekrenderer)
- [TextRenderer](#textrenderer)
